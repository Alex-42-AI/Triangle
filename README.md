# Triangle
Takes as input what the user knows about a triangle, when it comes to its sides, angles and heights, and returns everything, that can possibly be known about it, i.e. values in the triangle, as well as coordinates of its points, the triangle center point, the outer circle center point and the inner circle center point.
Input is taken as instances of "{variable_name}={value}" (spaces don't matter), separated by ", ". Variable names are a, b and c for the sides, A, B and C for the angles and h_a, h_b and h_c for the heights.
